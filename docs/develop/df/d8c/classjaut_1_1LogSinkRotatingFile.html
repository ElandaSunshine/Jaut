◆ eventLogRotated

Event<LogRotatedHandler, juce::CriticalSection> jaut::LogSinkRotatingFile::eventLogRotated

This event will be raised whenever the log file was rotated.
Note that, while adding events is thread-safe, the event being raised is not necessarily, as this will always be raised on the thread the log messages are processed on.

If Options::logRotationOptions::fileManagementBehaviour is set to LogRotationManager::FileManagementBehaviour::Delete, the event's parameter is an invalid file.
