Detailed Description

template<int BufferSize = 512, class CriticalSection = juce::DummyCriticalSection>
class jaut::LogWorkerSimple< BufferSize, CriticalSection >

The simple log worker which works synchronously to the thread the logger is owned by.


Do note that the logging thread will also be responsible for processing the messages.

Template Parameters
BufferSizeThe size of the buffer for log messages
CriticalSectionA lock that allows multiple threads to log messages
