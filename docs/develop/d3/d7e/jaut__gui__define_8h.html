#define JAUT_GUI_COLOUR_NAMESPACE   0x02

Config: JAUT_GUI_COLOUR_NAMESPACE

Specifies the JAUT namespace for colours, if for whatever reason you are not happy with the default, you can set it here to a different value. Do note though, that changing this, might make it incompatible in situations where colours are serialised and shared.
