Detailed Description

template<class T>
class jaut::NonNull< T >

Allows to pass a pointer that should not be assigned nullptr.

Note that if pointers that were nullptr assigned prior to passing, this will make a runtime check and responds correspondingly depending on the mode that was set with JAUT_CORE_NONNULL_HANDLE_NULLPTRS.

There is three possible approaches
Key Description
Throwing: 0 (default) Throws if a nullptr was encountered
Terminating: 1 Terminates if a nullptr was encountered
Ignoring: 2 Ignores the nullptr and continues as though nothing has happened

Note that for smart pointers, this does not make sure they can't be set to nullptr upon deletion or moving, so you will need to handle these cases carefully yourself.
