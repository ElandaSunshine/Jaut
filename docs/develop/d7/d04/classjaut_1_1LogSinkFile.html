◆ print()

void jaut::LogSinkFile::print ( const LogMessage &  message)
overridevirtual

Prints the message. If the sink manages a buffer, this should add the message to the buffer instead of directly outputting it.

Note that, if there is a buffer for this sink, the buffer should not be managing mere references to the data; it is not said the objects will still exist at the point of flushing.
