Detailed Description

A simple and lightweight Logger implementation that aims at being a bit more advanced than the juce one. The implementation of this logger is heavily inspired by Apache's log4j and gabime's spdlog, so you will find some similarities here and there.

Do note that loggers are immutable, so any options you give it should only be changed through re-constructing the logger.
