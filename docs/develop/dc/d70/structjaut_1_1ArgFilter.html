Detailed Description

template<class Predicate, class ... Args>
struct jaut::ArgFilter< Predicate, Args >

Provides a filter for variadic function templates, which allows, based on the given predicate, to filter out a set of arguments and process them whatever way needed.

Template Parameters
PredicateThe predicate to filter out the arguments
ArgsThe argument list of all arguments
