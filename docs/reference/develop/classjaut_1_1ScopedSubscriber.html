Detailed Description

template<class Handler>
class jaut::ScopedSubscriber< Handler >

A simple RAII event subscriber interface, that let's you automatically subscribe to and unsubscribe from an event.



Example:

void function()
{
Event<MyHandler> myEvent;
{
// Here the event will be subscribed to
ScopedSubscriber subscriber(myEvent, jaut::makeHandler(...));
} // Here the event will be unsubscribed from
}
