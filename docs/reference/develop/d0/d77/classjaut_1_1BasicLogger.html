Detailed Description

template<class Worker>
class jaut::BasicLogger< Worker >

A proper logging implementation that aims at being a bit more advanced than the juce native one.

The implementation of this logger is heavily inspired by Apache's log4j and gabime's spdlog, so you will find some similarities here and there.

Do note that loggers are immutable, so any options you give it can only be changed when re-constructing the logger.
