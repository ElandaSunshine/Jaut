staticconstexprnoexcept

Calls a functor exactly once with all types.

Other than forEach, this does not call the functor individually for each single argument but rather all together which allows for parameter pack expansions.

The functor must be a functor like:

using MyTypeList = jaut::TypeArray<int, double, float>;
struct MyFunctor
{
template<class ...Types>
void operator()(int test)
{
// Do stuff with Types
}
}
// Later in code
