Detailed Description

A worker is a separate layer to a jaut::BasicLogger, used to actually transport messages to the logging sinks.

Different worker implementations might handle log distribution differently.

The worker will act as an advanced buffer that also cares about processing incoming messages so that every sink gets what it deserves.
