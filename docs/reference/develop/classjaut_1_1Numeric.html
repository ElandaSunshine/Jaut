Detailed Description

template<class T, NumericCheck... CheckFlags>
class jaut::Numeric< T, CheckFlags >

An numeric base class for classes that need to be arithmetically operable.

It can also be used as a direct wrapper for numeric types.

Note that, if for integral types overflow-checking is enabled, this will incur a heavy runtime-cost. As every modification to this object needs to be first converted to a juce::BigInteger object which has to dynamically allocate memory for it to be able to store bigger numbers.
