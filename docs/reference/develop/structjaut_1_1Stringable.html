Detailed Description

template<class T>
struct jaut::Stringable< T >

Defines a customisable template specialisation provider that allows you to convert objects to juce::String.



To define your own object you need to either specialise this template for your object and a corresponding toString(const Object&) static function or have a toString() method inside your object which returns a to juce::String convertible type.

You will only need to specialise your own type if the type is not implicitly convertible to juce::var or juce::String.
If no other way of representing this object was found, it will print the name of the class and its address in memory. (ns::MyClass@0x0000...)

Additionally, you can define your own way how type names should be serialised with a simple static name() function which returns a string with the name.
Otherwise it will try to derive the type name with jaut::getActualTypeName() which will try to get the name from its typeid.

The major intent behind this class is debugging and logging, but you can use it for anything you like.
Just note that, for the most part, this does not return a representation that is usable for JSON or any other parser, but instead just a visual guideline helping in printing out the most important parts of an object.
This is heavily inspired by the way Java handles these things.
