◆ jdscoped

#define jdscoped   const auto JAUT_LINEISE_AUTOVAR(jaut_scoped, __LINE__) =

Allows defining a local scoped variable without a name.


This is useful for when you have something that is only affected by construction and destruction and you don't actually need access to the object.

e.g

jdscoped std::lock_guard(mutex);
